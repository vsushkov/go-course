## Добавить /debug/pprof

На отдельном порту запустить `net/http/pprof`. Получить через него heap профиль и визуализировать в SVG.
Пример хендлера, который расходует память (чтобы был показательный профиль):

```go
type Data struct {
    buffer []byte
}

var cache = make(map[int]*Data)

func leakyFunction() {
    for i := 0;; i++ {
        cache[i] = &Data{buffer: make([]byte, 1<<20)} // 1MB per iteration
        if i%10 == 0 {
            fmt.Printf("Leaked ~%d MB\n", i)
        }
        time.Sleep(100 * time.Millisecond)
    }
}

func handler(...) {
    go leakyFunction()
    select {}
}
```
