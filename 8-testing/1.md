# Small tasks

## Table tests для функций валидации

Покрыть тестами (table tests) функции для валидации входящих данных


## Тестирование HTTP хендлера

Используя `httptest.NewRecorder` и `httptest.NewRequest`, протестировать один из хендлеров,
проверяя и тело ответа, и статус.


## Тестирование репозитория

Напиши интеграционный тест на запись + получение одной записи из БД.
Не забудь подчистить после теста базу (либо transaction rollback, либо `TRUNCATE`).
